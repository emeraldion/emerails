Cookie::delete (   $name)
static

Deletes a cookie.

Sets a cookie to some date in the past: this may seem ridiculous, but is the only recommended way to cause the deletion of a cookie.

Warning
This function should be called only for cookies set with set using default values, otherwise they won't be deleted properly. In all other cases, set should be used.
Parameters
nameThe name of the cookie.

Sets the value and other parameters of a cookie.

The value of the cookie is added to the superglobal _COOKIE array in order to be readily available on the rest of the code.
